# 🛍️ Milano Imports

Milano Imports é um projeto de e-commerce desenvolvido com o objetivo de praticar desenvolvimento web e aplicar conhecimentos adquiridos durante a graduação em Análise e Desenvolvimento de Sistemas.

O projeto simula uma loja virtual de produtos importados, oferecendo uma interface moderna e intuitiva para navegação entre categorias, autenticação de usuários e carrinho de compras.

---

## 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- PHP

---

## ✨ Funcionalidades

- 🏠 Página inicial responsiva
- 👤 Sistema de Login
- 📝 Cadastro de usuários
- 🛒 Carrinho de compras
- 👕 Catálogo de produtos
- 📂 Organização por categorias
- 📖 Página Sobre
- ❓ Central de Ajuda

## 💻 Como executar

1. Clone este repositório

```bash
git clone https://github.com/Brenno-create/milano-imports.git
```

2. Coloque a pasta do projeto dentro do diretório **htdocs** do XAMPP.

3. Inicie os serviços **Apache** e **MySQL**.

4. Acesse:

```
http://localhost/milanoimports
```
